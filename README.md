# Setup
1. Clone the repo
2. Move to that repo.
3. Update the username and other fields if neccessary in config.json file.  
3. Execute 'npm install' 
4. Execute 'node server.js'

Open localhost:8080 and start writing.

For saving the entry press "Ctrl + Shift + s"
